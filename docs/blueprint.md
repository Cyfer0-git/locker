# **App Name**: Locker

## Core Features:

- Credential Storage: Securely store website URLs, usernames, and passwords using AES encryption.
- Canned Message Templates: Create and save reusable message templates for quick access and easy copy/pasting.
- AES Encryption: Utilize AES encryption to ensure sensitive data is securely stored and protected within the application. Encrypt and decrypt on the client side with a master password.
- Data search: Filter data to easily find your credential and messages
- Dynamic Content Generation (AI): When a message is viewed or copied, this tool allows to generate some dynamic content based on what the user provides. This way, canned messages can be customized using some generative AI based on what is actually on the clipboard.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) for a sense of security and trustworthiness.
- Background color: Very light gray (#F9FAFB) for a clean and unobtrusive backdrop.
- Accent color: Sky blue (#38BDF8) to highlight interactive elements and provide visual interest.
- Body and headline font: 'Inter' (sans-serif) for a modern, readable, and neutral aesthetic.
- Use simple, consistent icons from a set like Heroicons or Font Awesome to represent actions and data types.
- Employ a clean, card-based layout to organize credentials and messages, with clear visual separation between items.
- Incorporate subtle transitions and feedback animations to enhance user interaction, such as fade-in effects when loading data or a brief highlight on copied text.
